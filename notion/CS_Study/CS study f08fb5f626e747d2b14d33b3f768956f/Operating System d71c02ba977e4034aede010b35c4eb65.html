Scheduler

Basic Approach

  • Non-preemtive
    • The scheduler waits for the running process to voluntarily yield the CPU
    • Processes should be cooperative
  • Preemptive
    • The scheduler can interrupt a process and force a context switch
    • Issues :
      • What if preempting on the midst of updating shared data?
      • What if preempting in a system call

Terminology

  • Workload : A set of job descriptions
  • Scheduler
  • Metric : Measurement of scheduling quality

Methodology

  • Non-preemtive
    • FIFO
    • SJF
  • Preemtive
    • STCF (Shortest Time-to-Completion First)
    • Round Robin
      • Give each job a time slice
      • Improve response time, overhead for switching
    • MLFQ ( Multi-Level Feedback Queue) : Priority - nice value
      • give process nice value
